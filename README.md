# trader
This Machine Learning Trading model uses HistGradientBoostingClassifier to perdict both 1 hour and 1 day price trends and puts them into states that the model can recognize. 

Currently It fetches 5 years of past data and feeds it to the 1 day price model.

The first prototype model has a 37% accuracy, but has beat the market/SPY 2/2 so far.
